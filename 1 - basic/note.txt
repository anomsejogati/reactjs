ReactJs

SESI 1
- Install React JS
    sudo npm install -g create-react-app
- Create Project
    create-react-app nama_project


SESI 2
- StateLess Component & StateFull Component
    StateLess Component = functional component
    StateFull Component = class component

SESI 3
- Memisahkan StateLess Component & StateFull Component di folder terpisah
    StateLess Component => component
    StateFull Component => container
